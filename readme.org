* cook recipes
This is (going to be) my collection of recipes, making use of [[https://github.com/abo-abo/cook][cook]]. To install,
** Installation
#+BEGIN_SRC bash
pip install pycook
git clone https://github.com/guanyilun/.cook.d ~/.cook.d
#+END_SRC

** Recipes examples

#+BEGIN_SRC 
ps aux | awk '{print $2}' | cook :plot show
